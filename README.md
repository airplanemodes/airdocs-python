# Introduction

> An awesome project.

The Python documentation that feels like an air and really easy to navigate through it.

### Installation

For viewing the website locally first install `docsify`:

```bash
$ npm i docsify-cli -g
```

Then:

```bash
$ git clone https://github.com/airplanemodes/airdocs-python
$ cd airdocs-python
$ docsify serve
```
