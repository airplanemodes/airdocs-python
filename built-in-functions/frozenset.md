# frozenset()

Return a new [`frozenset`](/built-in-types/frozenset/) object, optionally with elements taken from **iterable**.

### Syntax

```python
frozenset()
frozenset(iterable=set())
```

### Examples

```python
>>> vowels = frozenset(["a", "e", "i", "o", "u"])
>>> vowels # frozenset({'e', 'o', 'i', 'u', 'a'})
```
