# set()
Return a new set object, optionally with elements taken from **iterable**. If **iterable** is not specified, a new empty set is returned.

### Syntax
```python
set()
set(iterable)
```

### Examples
```python
set('foobar') # {'r', 'b', 'o', 'f', 'a'}
set(['a', 'b', 'c']) # {'c', 'b', 'a'}
```
