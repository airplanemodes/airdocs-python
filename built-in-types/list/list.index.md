# list.index()

`>>> list.index.__doc__`: Return first index of value. Raises [`ValueError`](/exceptions/ValueError.md) if the value is not present.

### Syntax

```python
sequence.index(x)
sequence.index(x, i)
sequence.index(x, i, j)
```

### Example

```python
numbers = [1, 2, 3]
numbers.index(2) # 1
```
