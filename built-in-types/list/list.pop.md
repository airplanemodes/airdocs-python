# list.pop()

Retrieves the item at index **i** and also removes it from list.

```python
numbers = [1, 2, 3]
numbers.pop(1)
print(numbers) # [1, 3]
```

If **i** is not given it retrieves the last item and also removes it from list. 

```python
numbers = [1, 2, 3]
numbers.pop()
print(numbers) # [1, 2]
```
