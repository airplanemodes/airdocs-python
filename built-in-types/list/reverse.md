# list.reverse()

`>>> list.reverse.__doc__`: Reverse **IN PLACE**.

### Syntax

```python
list.reverse()
```

### Examples

```python
numbers = [1, 2, 3]
numbers.reverse()
print(numbers) # [3, 2, 1]
```

There is an alternative way to reverse a list:

```python
>>> numbers = [1, 2, 3]
>>> numbers[::-1] # [3, 2, 1]
```
