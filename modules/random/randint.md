# random.randint()

Return a random integer *N* such that `a <= N <= b`.

Alias for `randrange(a, b+1)`.

### Syntax

```python
random.randint(a, b)
```

### Examples

```python
>>> random.randint(1, 10) # 3
>>> random.randint(1, 10) # 6
>>> random.randint(1, 10) # 1
>>> random.randint(1, 10) # 4
>>> random.randint(1, 10) # 10
>>> random.randint(1, 10) # 8
```
