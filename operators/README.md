# Operations
There is boolean operations - `or`, `and`, `not`.

Comparsion operations:
- `<` less than
- `<=` less than or equal
- `>` greater than
- `>=` greater than or equal
- `==` equal
- `!=` not equal
- `is` object identity
- `is not` negated object identity

Bitwise operations:
- `|` bitwise or
- `^` bitwise exclusive or
- `&` bitwise and
- `<<` shift left
- `>>` shift right
- `~` invert
