# Operators

There is assignment operator: `=`.

Arithmetic operators:
- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division
- `//` floor division
- `%` modulo
- `**` exponentiation

Boolean operators: `or`, `and`, `not`.

Comparsion operators:
- `<` less than
- `<=` less than or equal
- `>` greater than
- `>=` greater than or equal
- `==` equal
- `!=` not equal
- `is` object identity
- `is not` negated object identity

Bitwise operators:
- `|` bitwise or
- `^` bitwise exclusive or
- `&` bitwise and
- `<<` shift left
- `>>` shift right
- `~` invert
